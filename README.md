## Overview

This repository is a personal documentation library — a centralized place for technical references, development methodologies, and practical guides authored for real use. Guides are written for clarity and immediate applicability, not as theoretical exercises.

Content spans three domains: **version control** (Git/GitHub), **frontend development** (methodology and patterns), and **AI-augmented workflows** (vibe coding with LLMs). A bilingual Git guide (Arabic + English) is included as an interactive HTML experience.

**Format:** Interactive HTML (bilingual)

## <img src="https://api.iconify.design/lucide:terminal.svg?color=%238A2BE2" width="20" height="20" align="center" /> Usage

### Read Markdown Guides

Open any `.md` file directly in GitHub — they render automatically.

### Open the Interactive Git Roadmap

```bash
# Clone the repository
git clone https://github.com/AhmedTyson/Guides.git

cd Guides

# Open English version
start git-roadmap/index.html

# Open Arabic version
start git-roadmap/arabic.html
```

> Guides are continuously updated as new workflows and methodologies are adopted.
